Basic Matching Operators

. Matches any single character except newline.
^ Matches start of string or line.
$ Matches end of string or line.
\ Escapes special characters.

Quantifiers

* Matches last character 0 or more times.
+ Matches last character 1 or more times.
? Matches last character 0 or 1 time.
{n} Matches exactly n occurrences of last character.
{n,} Matches n or more occurrences of last character.
{n,m} Matches between n and m occurrences (incl.) of last character.

Character Classes

[abc] Character class: matches any one of the characters.
[^abc] Negated character class: matches any character except those listed.
- Specifies a range within a character class.
\d Matches any digit (equivalent to [0-9]).
\D Matches any non-digit (equivalent to [^0-9]).
\w Matches any word character [A-Za-z0-9_].
\W Matches any non-word character.
\s Matches any whitespace character.
\S Matches any non-whitespace character.

Grouping and Capturing

() Capturing group: groups and captures matched content.
(?:) Non-capturing group: groups without capturing.
\n Backreference to the nth capture group.

Alternation

| Alternation (OR operator).

Lookarounds (Zero-Width Assertions)

(?=) Positive lookahead: asserts what follows the current position.
(?!) Negative lookahead: asserts what doesn't follow the current position.
(?<=) Positive lookbehind: asserts what precedes the current position.
(?<!) Negative lookbehind: asserts what doesn't precede the current position.

Modifiers (Flags)

i Case-insensitive matching.
g Global matching (find all matches).
m Multiline mode (^ and $ match start/end of each line).
s Dot matches newline.
u Unicode mode (enables full Unicode matching).
y Sticky mode (matches only from the last index).

Boundary Matchers

\b Word boundary.
\B Non-word boundary.
